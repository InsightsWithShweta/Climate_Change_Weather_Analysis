
# 🌍 Climate Change & Weather Analysis Dashboard

## Project Overview
This project analyzes climate and weather trends across Indian states from 2000 to 2025 using Python, SQL, and Power BI. It includes data cleaning, exploratory data analysis (EDA), interactive dashboards, forecasting, and actionable insights into temperature, rainfall, humidity, heat index, and climate patterns.

## Tools & Technologies
- Power BI
- Python (Pandas, Matplotlib)
- SQL (MySQL)

## Dataset
- India Climate & Weather Dataset (2000–2025)

## Key Features
- Interactive Power BI Dashboard
- Temperature Trend Analysis
- Average Rainfall by State
- Temperature Category Distribution
- Temperature & Rainfall Forecasting
- El Niño vs Rainfall Analysis
- Dynamic Filters & KPI Cards
- Actionable Climate Insights

## Dashboard Pages

### Page 1 – Analysis
- KPI Cards
- Temperature Trend Analysis
- Average Rainfall by State
- Temperature Category Distribution
- Interactive Filters
- Key Insights

### Page 2 – Forecasting 
- Temperature Forecast
- Rainfall Forecast
- El Niño vs Rainfall Analysis
- Forecast Summary

## Key Insights
- Average temperature shows an increasing trend over time.
- Rainfall patterns vary significantly across Indian states.
- El Niño events influence rainfall variability in several regions.
- Forecasting indicates a gradual increase in average temperature.
